# NetworkChess

What it does:
Backend API for an online chess game. Handles player authorizations, game logic, player sessions, games lifecycle, moves and statistics.
Provides a system of saves using PostgreSQL database.
(frontend repo : https://github.com/shmettaaa/NetworkChess-front)
# Stack:

C#(.NET), ASP.NET Core Web API, SignalR Library;
Entity Framework, PostgreSql.
